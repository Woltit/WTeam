  <i>Peer-to-peer marketplace for short-term item rentals</i>

***

## 📖 Про проєкт

**RentGo** — це двостороння веб-платформа (peer-to-peer маркетплейс) для короткострокової оренди речей між фізичними особами. Платформа з'єднує **власників**, які хочуть монетизувати своє майно, з **орендарями**, яким річ потрібна на обмежений час — дриль, намет, камера, велосипед та інше.

Проєкт розроблено в рамках навчальної програми зі спеціальності **«Інженерія програмного забезпечення»** в Національному університеті **«Києво-Могилянська академія» (НаУКМА)** та орієнтовано на ринок України.

> **Статус:** MVP — термін реалізації 4 тижні

***

## ✨ Ключові особливості

### 🤖 AI-асистент
Вбудований модуль на базі **Anthropic API** підбирає речі на основі текстового опису ситуації або поїздки користувача. Наприклад, достатньо написати _«їду на три дні в Карпати»_ — і система запропонує відповідні позиції.

### 📅 Календар бронювання
Інтерактивний календар перевіряє доступність речі в реальному часі та захищає від **подвійного бронювання** (_double booking prevention_).

### 🛡️ Система довіри (Trust Score)
Багаторівнева система довіри включає:

- **Верифікацію особистості** — ручна перевірка за паспортом
- **Репутаційний рейтинг** — накопичується з часом
- **Двосторонні відгуки** — власник та орендар оцінюють одне одного після кожної оренди

### 🖥️ Адміністративна панель
Інструмент для внутрішньої модерації: перегляд та схвалення оголошень, вирішення конфліктів між користувачами, статистика платформи.

## 🛠️ Технічний стек

### Backend

| Технологія | Призначення |
|---|---|
| **Golang (Gin)** | Основний API-сервіс, висока продуктивність |
| **Java (Spring Boot)** | Модулі бізнес-логіки, бронювання, Trust Score |
| **PostgreSQL** | Основна реляційна база даних |
| **Redis** | Кешування, управління сесіями |
| **Apache Kafka** | Асинхронна комунікація між сервісами |
| **Hibernate** | ORM для Java-компонентів |

### Frontend

| Технологія | Призначення |
|---|---|
| **React.js (TypeScript)** | SPA-інтерфейс |
| **Tailwind CSS** | Утилітарне стилізування |
| **Redux** | Глобальний стан застосунку |

### AI / ML

| Технологія | Призначення |
|---|---|
| **Anthropic API** | AI-асистент для підбору речей |

### Інфраструктура та CI/CD

| Технологія | Призначення |
|---|---|
| **AWS** | Хмарний деплой |
| **Docker / Docker Compose** | Контейнеризація сервісів |

## 🏗️ Архітектура

Платформа побудована за принципами **мікросервісної архітектури**:

- **API-шлюз** — маршрутизація запитів між сервісами
- **Go-сервіс** — публічний API, AI-інтеграція, оптимізація продуктивності
- **Spring Boot-сервіс** — бронювання, управління користувачами, Trust Score, адмін-панель
- **React SPA** — клієнтська частина, комунікує через **Swagger API-контракт**
***

## 🔐 Безпека

- **HTTPS** — шифрування всього трафіку
